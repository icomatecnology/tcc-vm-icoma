# 🌍 Máquina Nuvem - TCC Icoma

## Uma Iniciativa ESG em Ação

Bem-vindo ao projeto **Máquina Nuvem**! Somos uma iniciativa comprometida com **Sustentabilidade**, **Responsabilidade Social** e **Governança Corporativa**. Aqui, a tecnologia encontra o propósito.

---

## 💚 Nossa Missão ESG

Utilizamos infraestrutura em nuvem de forma inteligente e responsável para:

- **🌱 Sustentabilidade**: Otimizar recursos computacionais e reduzir consumo de energia
- **👥 Responsabilidade Social**: Democratizar acesso à tecnologia nuvem para comunidades
- **⚖️ Governança**: Transparência total no código e processos

---

## 🛠️ Stack Tecnológico

Desenvolvemos com tecnologias modernas e eficientes:

| Tecnologia | Uso | 
|-----------|-----|
| **Python + Flask** | Framework web robusto para backend escalável |
| **Nix** (36.5%) | Gerenciamento de ambiente declarativo |
| **HTML** (30.4%) | Interface e documentação web |
| **Shell** (17%) | Automação e scripts de sistema |

### 🔧 Foco Principal: Python + Flask

Este projeto utiliza **Python com Flask** como framework principal, permitindo desenvolvimento ágil, eficiente e sustentável de aplicações web modernas.
